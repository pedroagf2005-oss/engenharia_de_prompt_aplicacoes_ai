# engenharia_de_prompt_aplicacoes_ia 
repositório/ativadades da faculdade ou material de papiro 


 este repositório tem como finalidade de abordar A disciplina Engenharia de Prompt e Aplicações de Inteligência Artificial, que tem como objetivo capacitar os estudantes a compreender e aplicar técnicas avançadas de interação com modelos de IA. Em um cenário o qual a inteligência artificial se torna cada vez mais presente em diferentes áreas. Com isso, dominar a arte de formular prompts eficazes é essencial para explorar todo o potencial dessas tecnologias.
